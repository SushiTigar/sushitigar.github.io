XBOX ONE RETAIL TRAINING

eLearning Course

This course is intended to teach the main selling points of the Xbox One to Retail Associates so that they may successfully answer questions and sell it to customers. This course features animations and interactive quiz questions.

My Contributions
I wrote and storyboarded this course using Microsoft Word and PowerPoint. I worked with Graphic Designers and Web Developers to ensure that this course met Microsoft's needs.

Tools Used
Microsoft Word and PowerPoint were my main tools and Articulate Storyline was used to develop this course.
